## Dotfiles Installer

```just
Available recipes:
    apply             # Apply dotfiles
    arch-dependencies # Install arch-dependencies
    default           # List recipes
    install           # Install chezmoi
    setup_arch        # Run everything together on arch
    setup_mac         # Run everything together on mac
```

> [!NOTE]
Personal secrets are stored in [ProtonPass](https://proton.me/pass) and you'll
need the [ProtonPass CLI](https://protonpass.github.io/pass-cli/) installed.
